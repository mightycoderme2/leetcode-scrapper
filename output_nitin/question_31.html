31. Next Permutation

Implement next permutation , which rearranges numbers into the lexicographically next greater permutation of numbers.

If such an arrangement is not possible, it must rearrange it as the lowest possible order (i.e., sorted in ascending order).

The replacement must be in place and use only constantÂ extra memory.

Example 1: 

Input: nums = [1,2,3]
Output: [1,3,2]

Example 2: 

Input: nums = [3,2,1]
Output: [1,2,3]

Example 3: 

Input: nums = [1,1,5]
Output: [1,5,1]

Example 4: 

Input: nums = [1]
Output: [1]

Constraints:

  • 1 <= nums.length <= 100
  • 0 <= nums[i] <= 100

// @lc code=start
class Solution {
public:
    void nextPermutation(vector<int>& a) {
        //find a element that is greater than the previous
        int n=a.size(),l,k;
        for(k=n-2;k>=0;k--){
            if(a[k+1]>a[k]){
                break;
            }
        }
        //else reverse
        if(k<0){
            reverse(a.begin(),a.end());
        }
        //find a element that is greater than the element of k index
        else{
            for(l=n-1;l>k;l--){
                if(a[l]>a[k]) break;
            }
            //swap those element and reverse
            swap(a[k],a[l]);
            reverse(a.begin()+k+1,a.end());
        }
    }
};
