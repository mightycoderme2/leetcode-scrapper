371. Sum of Two Integers

Given two integers a and b , return the sum of the two integers without using the operators + and - .

Example 1: 

Input: a = 1, b = 2
Output: 3

Example 2: 

Input: a = 2, b = 3
Output: 5

Constraints:

  • -1000 <= a, b <= 1000 
/*
 * @lc app=leetcode id=371 lang=cpp
 *
 * [371] Sum of Two Integers
 */

// @lc code=start
class Solution {
public:
    int getSum(int a, int b) {
        int sum=a;
        while(b!=0){
            sum=a^b;
            b=(a&b)<<1;
            a=sum;
        }
        return sum;
    }
};
// @lc code=end
