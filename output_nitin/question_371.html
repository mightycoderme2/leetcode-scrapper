371. Sum of Two Integers

Given two integers a and b , return the sum of the two integers without using the operators + and - .

Example 1: 

Input: a = 1, b = 2
Output: 3

Example 2: 

Input: a = 2, b = 3
Output: 5

Constraints:

  • -1000 <= a, b <= 1000 
/*
 * @lc app=leetcode id=371 lang=cpp
 *
 * [371] Sum of Two Integers
 */

// @lc code=start
class Solution {
public:
    int getSum(int a, int b) {

        /*

        a = 0101
        b = 0111

        First Iteration:

        sum = a ^ b: 0101 ^ 0111 = 0010 (sum without considering carry)
        b = (a & b) << 1: (0101 & 0111) << 1 = 0101 << 1 = 1010 (carry)
        a = sum: a is now 0010

        Second Iteration:

        sum = a ^ b: 0010 ^ 1010 = 1000 (sum without considering carry)
        b = (a & b) << 1: (0010 & 1010) << 1 = 0010 << 1 = 0100 (carry)
        a = sum: a is now 1000

        Third Iteration:

        sum = a ^ b: 1000 ^ 0100 = 1100 (sum without considering carry)
        b = (a & b) << 1: (1000 & 0100) << 1 = 0000 (carry)
        a = sum: a is now 1100

        */

        int sum=a;
        while(b!=0){// add carry till it is zero
            sum=a^b; // sum without carrry
            b=(a&b)<<1; // carry
            a=sum;
        }
        return sum;


    }
};
