50. Pow(x, n)

Implement pow(x, n) , which calculates x raised to the power n (i.e., x n ).

Example 1: 

Input: x = 2.00000, n = 10
Output: 1024.00000

Example 2: 

Input: x = 2.10000, n = 3
Output: 9.26100

Example 3: 

Input: x = 2.00000, n = -2
Output: 0.25000
Explanation: 2-2 = 1/22 = 1/4 = 0.25

Constraints:

  • -100.0 < x < 100.0
  • -2 31 <= n <= 2 31 -1
  • -10 4 <= x n <= 10 4 
/*
 * @lc app=leetcode id=50 lang=cpp

// @lc code=start
class Solution {
public:
    double myPow(double x, int n) {
        int sign =1;
        if(x<0 && n%2!=0) sign =-1;
        return sign*exp(log(abs(x))*n);
    }
};
